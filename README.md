# Cosmetic Insights using Tableau  
### Navigating Cosmetics Trends and Consumer Behavior

## 📌 Overview
Cosmetic Insights using Tableau is a data analysis and visualization project aimed at uncovering trends, patterns, and consumer preferences in the cosmetics industry. The project leverages Tableau dashboards to transform raw data into actionable insights for better decision-making.

This project was developed as part of an internship, following a structured lifecycle including problem definition, project planning, and data visualization.

---

## 🎯 Problem Statement
Consumers often struggle to:
- Identify trending cosmetic products  
- Understand brand performance and pricing patterns  
- Make informed purchasing decisions due to lack of clear insights  

This project addresses these challenges by:
- Analyzing cosmetic product datasets  
- Visualizing consumer behavior and trends  
- Providing intuitive dashboards for easy interpretation  

---

## 🚀 Objectives
- Analyze cosmetics dataset to identify key trends  
- Understand customer preferences and product performance  
- Build interactive dashboards using Tableau  
- Enable data-driven insights for business and consumers  

---

## 🛠️ Tech Stack

## 📊 Key Features
